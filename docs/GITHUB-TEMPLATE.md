# Publicar este repo como plantilla de GitHub

1. **Sube el proyecto a GitHub** (crear repo nuevo y push, o conectar un repo existente).

2. **Activar como plantilla:**
   - Repo → **Settings** → **General**
   - En "Repository template", marca **Template repository**

3. **Uso por terceros:**
   - En la página del repo, botón **Use this template** → **Create a new repository**
   - El nuevo repo será una copia independiente; pueden personalizar `config/site.js` y el tema sin afectar al original.

4. **Recomendación:** en la descripción del repo indica que es una plantilla, por ejemplo:  
   *"Template: landing Albino + tema Shopify (Liquid). Usa 'Use this template' para crear tu copia."*
